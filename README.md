# 🍽️ QR-Based Smart Restaurant Management System

## 📌 Overview

The **QR-Based Smart Restaurant Management System** is a web-based application developed to automate restaurant operations such as order management, billing, and customer interaction. This system enhances efficiency, reduces manual work, and improves the overall dining experience through digital transformation.

---

## 🚀 Features

* 📱 QR Code-based Menu Access
* 🧾 Order Management System
* 🔄 Real-time Order Tracking
* 💳 Billing Management
* 👨‍🍳 Admin Dashboard for Order Control
* 📊 Efficient Data Handling with MySQL

---

## 🛠️ Tech Stack

* **Backend:** Python (Flask)
* **Database:** MySQL
* **Frontend:** HTML, CSS
* **Tools:** Git, GitHub

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```
git clone https://github.com/Nanekar123/QR-Based-Smart-Restaurant-System.git
```

### 2️⃣ Navigate to Project Folder

```
cd QR-Based-Smart-Restaurant-System
```

### 3️⃣ Create Virtual Environment

```
python -m venv venv
```

### 4️⃣ Activate Virtual Environment

**Windows:**

```
venv\Scripts\activate
```

### 5️⃣ Install Dependencies

```
pip install -r requirements.txt
```

### 6️⃣ Run the Application

```
python app.py
```

### 7️⃣ Open in Browser

```
http://127.0.0.1:5000/
```
